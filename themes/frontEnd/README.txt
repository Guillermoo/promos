# bigbase-shop-advanced
- Bootstrap E-Shop Template
- Version: 1.0 from 22nd september 2012
- Author: BigBase

# Description
This Bootstrap e-shop template is an advanced e-shop template designed using Bootstrap and some optional javascript libraries. It is compatible with any of the themes from bootswatch making it easy to change colors and style. Of course you can easily add components as it is based on the bootstrap template.

## Libraries
The e-shop makes use of the following javascript libraries:
- jQuery 1.8.1
- jQuery Nivo Slider 3.1
- jQuery Colorbox 1.3.19
- Modernizr 2.6.2
- respond.js 1.1.0
- Google Maps 3

## Responsiveness
This template includes a responsive and a fluid version. Of course by disabling these, the layout can also be fixed.

## Pages
- Index/home
- Category overview
- Product overview/search result
- Product detail
- Contact page
- Sitemap page
- Order process / shopping cart
- Multiple column text pages (1, 2, 3... columns)

## Features
- 7 different page layouts
- Lightbox images
- Advanced slideshows (not only default bootstrap)
- Search form
- Login form
- Address tags
- Dropdown menu
- Multiple item/product list styles
- Social media button
- Facebook like button
- Shopping cart overview in menu
- Product list filter
- Complete multiple steps order page, extendable with more steps
- HTML5/CSS3 W3C compatible
- Simple image rollover script
- Google maps integration
